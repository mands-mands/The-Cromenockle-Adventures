## 📖 Sobre o Jogo

Durante um acampamento escolar, **Ethan** se perdeu em uma floresta misteriosa.
Ao anoitecer, criaturas sombrias começaram a surgir entre as árvores.
Ao longe, ele avistou uma torre vigia. Talvez alguém pudesse ajudá-lo...

Colete os **3 pedaços do mapa** espalhados pelas plataformas e chegue à torre
antes que os fantasmas te alcancem!

---

## 🎮 Controles

| Tecla | Ação |
|-------|------|
| `← →` | Mover Ethan |
| `ESPAÇO` | Pular |
| `ESC` | Sair |

---

## 🕹️ Como Jogar

- Colete os **3 pedaços do mapa** espalhados pelas plataformas
- Desvie dos **fantasmas** — cada toque remove 1 coração
- Com os 3 pedaços coletados, chegue até a **torre vigia** para vencer
- Você tem **3 corações** — cuidado!

---

## 🛠️ Tecnologias

- Python 3.11
- Pygame 2.6.1

## 📁 Como Executar

### Pelo código fonte
```bash
# Clone o repositório
git clone https://github.com/mands-mands/The-Cromenockle-Adventures

# Instale as dependências
pip install pygame

# Execute o jogo
python main.py
```
